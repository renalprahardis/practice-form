## Getting Started

### Installation
1. Clone the repo
   ```sh
   git clone https://github.com/renalprahardis/practice-form
   ```
2. Install NPM packages
   ```sh
   npm install
   ```

## Running
* Run global test headless
   ```sh
   npm run test 
   ```

* Run with open browser
   ```sh
   npm run open
   ```
